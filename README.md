# Steps to Create porject on your own:

## Initializing the project

> yarn init -y

> yarn tsc --init

## Basic Setup

> yarn add -D typescript

> yarn tsc --init --rootDir src --outDir dist

> yarn add inversify reflect-metadata --save

## Compiling & Running

> yarn install

> yarn start
